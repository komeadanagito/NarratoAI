# 视频素材边框

`border_mode=asset` 会从本目录随机选择一张 `.png`，缩放到视频当前画布后叠加。
该过程不会裁剪、缩放或填充原视频，输出分辨率和宽高比保持不变。

替换素材时请遵守以下约定：

- PNG 必须带 Alpha 透明通道，中心主体区域必须完全透明。
- 推荐画布为 `1920×1080`；运行时会适配横屏、竖屏及其他尺寸。
- 装饰内容应限制在画布四周约 2% 的区域内，避免遮挡主体和字幕。
- 文件名可以任意修改；服务会读取本目录下所有非符号链接 PNG。
- `sources/` 保存可编辑的 SVG 源文件，不会被运行时加载。
